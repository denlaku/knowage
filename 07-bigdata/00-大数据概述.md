Hadoop1.0和2.0：

![](./imgs/hadoop_1_vs_2.png)



+ **HDFS**（Hadoop Distributed File System） 分布式存储系统
    提供了高可靠性、高扩展性和高吞吐率的数据存储服务 
+ **YARN**（Yet Another Resource Negotiator） 资源管理系统
    负责集群资源的统一管理和调度 
+ **MapReduce**  分布式计算框架
     具有易于编程、高容错性和高扩展性等优点 

**Hadoop2.0层级关系**

+ MapService 分布式计算层
+ YARN 集群资源管理层
+ HDFS分布式存储层

#### Hadoop构成 ：

**1、HDFS特点** 

+ 良好的扩展性
+ 高容错性 
+ 适合PB级以上海量数据的存储 

**2、YARN（资源管理系统）**

+ YARN是什么 
  Hadoop 2.0新增系统 
  负责集群的资源管理和调度 
  使得多种计算框架可以运行在一个集群中 

+ YARN的特点 
  良好的扩展性、高可用性 
  对多种类型的应用程序进行统一管理和调度 
  自带了多种多用户调度器，适合共享集群环境
